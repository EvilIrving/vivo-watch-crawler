# 概述

更新时间：2023-10-31 16:42:47


蓝河应用的电话能力模块的多功能性使其成为实现通信和互动的强大工具。它为应用提供了多种通信方式，可以通过短信进行信息传递以及通过电话进行语音通话，满足不同用户需求。这对于社交应用、通讯工具、客户服务和多种应用场景都具有广泛的适用性。


### 子模块介绍


| 模块 | 简述 |
| --- | --- |
| 短信 | 提供了短信服务集成和短信功能实现支持 |
| 打电话 | 提供了电话通信支持，以满足用户的通话需求 |
