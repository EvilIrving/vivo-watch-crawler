## 接口定义


### screen.getScreenOffTime()


获取熄屏时间


##### 参数：


无


#### 返回值：


| 类型 | 说明 |
| --- | --- |
| Number | 1-999 秒 |


#### 示例：


```

screen.getScreenOffTime()
```

复制代码
### screen.getAodStatus()


获取 AOD 状态


> 
> AOD：Always on Display，即不点亮整块屏幕的情况下，控制屏幕局部亮起，将一些重要的信息一直显示在屏幕上。
> 
> 
> 


#### 参数：


无


#### 返回值：


| 类型 | 说明 |
| --- | --- |
| Number | 0: AOD 关闭; 1: AOD 打开 |


#### 示例：


```

screen.getAodStatus()
```
