# 概述

更新时间：2023-10-30 17:10:04


提供用于实现页面元素动画效果的 UI 组件，通常用于增强用户交互体验，吸引用户的注意力，提高页面的美观度和可视化效果。


## 动画组件介绍


| 组件 | 简述 |
| --- | --- |
| svg-container | 渲染 svg 图片，可以动态修改 svg 属性 |
| image-animator | 图片帧动画播放器 |
| animated-vector | animated-vector 组件，用于解析渲染安卓 xml 动画资源 |
