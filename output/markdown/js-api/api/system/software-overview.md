# 概述

更新时间：2023-10-31 19:37:39


蓝河应用的基础软件能力为应用提供了基础的软件功能支持，包括系统设置、输入法、解压缩、解包和序列化等，可以帮助开发者更加高效、快速地进行软件开发和计算机操作


## 子模块介绍


| 模块 | 简述 |
| --- | --- |
| 系统设置 | 提供获取系统设置能力 |
| 输入法 | 为输入法应用提供向 input 组件写入数据能力 |
| 解压缩 | 提供解压本地文件能力 |
| 解包 | 提供解包能力 |
| 序列化 | 提供序列化数据，反序列化数据能力 |
