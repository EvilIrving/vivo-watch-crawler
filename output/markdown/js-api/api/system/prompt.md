## 接口定义


### prompt.showToast(OBJECT)


显示 Toast


#### 参数：


| 参数名 | 类型 | 必填 | 说明 |
| --- | --- | --- | --- |
| message | String | 是 | 要显示的文本 |
| duration | Number | 否 | 0 为短时，1 为长时，默认 0 |


#### 示例：


```

prompt.showToast({
  message: 'message',
})
```
