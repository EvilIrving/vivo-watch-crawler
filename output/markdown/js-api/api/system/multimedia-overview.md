# 概述

更新时间：2023-10-29 18:27:39


蓝河应用多媒体能力支持音频业务的开发，提供音频相关的功能，主要包括音频播放、音频流式播放、录音、音量管理等。


## 子模块介绍


| 模块 | 简述 |
| --- | --- |
| 音频 | 提供音频播放能力 |
| 多媒体 | 提供音频播放、音频流式播放、录音能力 |
| 音频管理 | 提供音量设置，音量获取、监听等音频管理能力 |
| 录音 | 提供录音能力 |
